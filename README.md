# Wonderful Editor — Ультимативный Текстовый Редактор для Писателей

Wonderful Editor — это продвинутый современный текстовый редактор для Android, спроектированный специально для авторов книг, статей, сценариев и создателей контента. Приложение написано на Kotlin с использованием Jetpack Compose для обеспечения плавного, отзывчивого интерфейса с 60 FPS и красивыми микроанимациями.

## Основной функционал

*   **Продвинутое форматирование текста**: жирный, курсив, подчеркнутый, зачеркнутый, динамическое изменение размера шрифта, абзацные отступы, настройка межстрочного и межбуквенного интервала.
*   **Выравнивание текста**: возможность выравнивать текст по левому краю, по правому краю, по центру и по ширине страницы.
*   **Работа со шрифтами**: выбор из встроенных шрифтов и возможность импортировать собственные файлы шрифтов (`.ttf`/`.otf`) напрямую из хранилища смартфона.
*   **Блочный макет (Notion-style)**: поддержка блоков заголовков (H1, H2, H3), абзацев, картинок и параллельного двухколоночного текста (для параллельного письма или сравнения).
*   **Вставка изображений**: импорт картинок в текст с настройкой их размера (в процентах от ширины экрана) и выравнивания. Картинки копируются в приватную директорию приложения для надежного сохранения путей.
*   **Автосохранение и ручное сохранение**: мгновенное автоматическое сохранение после любого изменения содержимого, а также кнопка принудительного сохранения.
*   **Управление файлами**: переименование файлов, быстрое создание дубликатов (копий) и безопасное удаление.
*   **Библиотека файлов**: главный экран с сортировкой по новизне (сначала новые) или наоборот (сначала старые), строкой поиска по названию и интерактивным баннером статистики.
*   **Баннер статистики**: отображает общее количество файлов, суммарное число слов и букв, а также название и точное время последнего редактирования файла.
*   **Нумерация страниц**: полностью настраиваемая нумерация страниц (сверху/снизу слева/справа) с возможностью задать размер, цвет, шрифт и начертание для номеров страниц в экспортированных файлах.
*   **Мультиформатный экспорт**: сохранение документов в форматы **PDF**, **DOCX (Microsoft Word)**, **HTML** и **TXT** с помощью стандартного Android-диалога сохранения.
*   **Система корзины**: файлы удаляются сначала в корзину. В корзине доступен просмотр характеристик, выборочное или массовое восстановление, а также безвозвратное удаление с подтверждением.
*   **Адаптивный дизайн**: приложение подстраивается под экран любого смартфона, поддерживает портретный и альбомный (горизонтальный) режимы экрана.
*   **Абсолютно темная тема (Amoled-like)**, светлая тема и желтая тема (эффект старой книги/бумаги).
*   **Локализация**: поддержка русского, английского и украинского языков с переключателем в реальном времени.
*   **Раздел "О приложении"**: содержит сведения о бета-версии (0.1 Beta), авторе ("Файги") и кликабельной иконке Telegram, ведущей на канал `t.me/FeigeTG`.

## Как скомпилировать приложение (APK) через GitHub Actions

Вам **не нужно скачивать локально** никакие Android SDK, Gradle или компиляторы. Сборка будет проходить полностью в облаке GitHub:

1.  **Создайте новый репозиторий на GitHub**:
    *   Зайдите в свой профиль на [GitHub](https://github.com) и нажмите кнопку **New** (Новый репозиторий).
    *   Назовите его, например, `WonderfulEditor` и нажмите **Create repository**.
2.  **Загрузите все файлы из этой папки в ваш репозиторий**:
    *   Вы можете сделать это через git-консоль или просто перетащив все файлы в веб-интерфейс GitHub.
    *   *Важно*: обязательно залейте скрытую папку `.github` и файл `build.yaml`.
3.  **Автоматический запуск сборки**:
    *   Как только файлы окажутся на GitHub, запустится автоматический скрипт сборки, прописанный в `.github/workflows/build.yml`.
    *   Вы можете увидеть статус сборки во вкладке **Actions** вашего репозитория.
4.  **Скачивание готового APK**:
    *   Перейдите во вкладку **Actions**.
    *   Нажмите на текущий запущенный или уже завершенный рабочий процесс (например, `Build Android APK`).
    *   Прокрутите открывшуюся страницу в самый низ до блока **Artifacts** (Артефакты).
    *   Нажмите на название артефакта **Wonderful-Editor-APK**. Скачается ZIP-архив.
    *   Распакуйте скачанный ZIP-архив на вашем телефоне или ПК — внутри будет лежать готовый к установке файл `app-debug.apk`!
